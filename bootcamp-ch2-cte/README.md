# Chapter 2 :: Contention

Module for demonstrating patterns for dealing with workload contention.

Key topics:

- Avoiding read-modify-write contention magnet
- Serializable vs read-committed isolation levels
- Pessimistic locks on top of isolation levels
- Historical reads (exact and bounded staleness)
- Modifying CTEs

## Running Tests

This script will present a menu of available test:

    ./run-test.sh
